# Verba icon assets

These SVG assets are extracted from the global floating UI flow design and are intended for the Flutter client.

## Icons

- `verba-pill-mark.svg` - idle floating pill
- `verba-start.svg` - start listening
- `verba-stop.svg` - stop listening
- `verba-audio-source.svg` - WASAPI/system audio status
- `verba-subtitle-pane.svg` - open sliding subtitle pane
- `verba-dashboard.svg` - open data console
- `verba-correction.svg` - corrected subtitle marker
- `verba-font-decrease.svg` - decrease subtitle size
- `verba-font-increase.svg` - increase subtitle size
- `verba-collapse.svg` - collapse panel
